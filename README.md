##  Article app Frontend

---

## Installation

1. Clone or download this repository.
2. Install dependencies: npm install.

---

## Usage

1. To run application use: "npm run dev";

This will start server at http://localhost:5173.

2. 
- Click an article in the sidebar to view its content.
- Click "Create Article" to add a new one.
- Fill in the title and content, then submit.

---

## Backend API
- Make sure the backend is running at http://localhost:3001
- The frontend fetches articles from the backend API.
